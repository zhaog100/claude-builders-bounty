# n8n Workflow: Weekly Dev Summary

Automated weekly development summary using Claude API.

## Quick Start

1. Import `weekly-dev-summary.json` into n8n
2. Configure credentials (GitHub Token, Claude API Key)
3. Activate workflow

## Requirements

- n8n instance
- GitHub Personal Access Token
- Claude API Key

## Schedule

Every Monday at 9:00 AM

## Output

Formatted Markdown summary of weekly commits.
