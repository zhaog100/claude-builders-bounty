## 📚 学习要点总结

### 1️⃣ Home Assistant 网络模式

**核心知识**:
- **必须使用 `network_mode: host`**
- **原因**: 支持 mDNS/UPnP 设备发现
- **限制**: Bridge 模式无法自动发现设备

**实际应用**:
- Docker Compose 配置
- README 说明最佳实践
- 提供 Bridge 替代方案（注释）

### 2️⃣ Mosquitto 安全配置

**核心知识**:
- `allow_anonymous false` - 禁用匿名
- `password_file` - 密码认证
- `acl_file` - 访问控制
- WebSocket 支持（端口 9001）

**实际应用**:
- Docker 容器密码创建
- ACL 规则配置
- 持久化存储

### 3️⃣ macOS 系统优化

**核心知识**:
- `mediaanalysisd`: 照片/视频分析，高 CPU
- 小部件: macOS 自动保护，需 LaunchAgent
- 优化工具: 脚本 + 定时任务

**实际应用**:
- 创建 `disable-widgets.sh`
- 配置 LaunchAgent
- 内存优化 +716MB
